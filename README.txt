Manual install and run
1. Navigate to project directory: `cd cs325_portfolio_project`
2. Install virtual environment: `python3.8 -m venv .venv`
3. Activate virtual env: `source .venv/bin/activate`
4. Install requirements: `pip install -r requirements.txt`
5. Run program: 'python main.py'

Install script
1. Navigate to project directory: `cd cs325_portfolio_project`
2. Run script: `chmod +x ./install.sh && ./install.sh`
3. Run program: `python main.py`
